# Employee Management System

## Project Overview

Employee Management System is a Spring Boot based REST API application developed to manage employee records securely. The application provides CRUD operations, JWT authentication, role-based authorization, pagination, sorting, transaction management, and exception handling.

## Features

* Employee CRUD Operations
* JWT Authentication
* Spring Security Integration
* Role-Based Access Control (ADMIN, USER)
* Global Exception Handling
* Pagination and Sorting
* Transaction Management
* MySQL Database Integration
* RESTful APIs
* DTO Layer Implementation
* Validation using Bean Validation

## Technologies Used

* Java 17
* Spring Boot
* Spring Security
* Spring Data JPA
* Hibernate
* MySQL
* Maven
* JWT
* Eclipse IDE
* Git & GitHub

## API Modules

### Authentication APIs

* Register User
* Login User
* Generate JWT Token

### Employee APIs

* Create Employee
* Get Employee By ID
* Get All Employees
* Update Employee
* Delete Employee

## Security

The application uses JWT Authentication and Role-Based Authorization.

Roles:

* ADMIN
* USER

## Database

MySQL database is used for storing employee and user information.
